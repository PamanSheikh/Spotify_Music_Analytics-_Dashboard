# 🎧 Spotify Music Analytics Dashboard

## 📌 Overview
This project is an interactive music analytics dashboard built using Plotly Dash. It allows users to explore Spotify track data through dynamic and interactive visualizations. The dashboard provides insights into genres, artists, and audio features like danceability and energy.

---

## 🚀 Features
- Interactive filters (Genre & Popularity)
- Genre distribution (Pie Chart)
- Danceability vs Energy analysis (Scatter Plot)
- Top 10 artists visualization (Bar Chart)
- Song duration distribution (Histogram)
- Dark-themed UI using Bootstrap
- Automatic dataset download using KaggleHub

---

## 🛠️ Technologies Used
- Python  
- Dash (Plotly Dash)  
- Plotly Express  
- Pandas  
- Dash Bootstrap Components  
- KaggleHub
